Assume \(\varepsilon \ge 0\) and \(\operatorname {Bias}(S) \ge 2\sqrt{2} - \varepsilon \). Then there exist isometries \(V_A\), \(V_B\) and a junk state \(\Phi _{\mathrm{junk}}\) such that

\[ \left\lVert \mathrm{regSwap}\bigl((V_A \otimes V_B)(A_0 \otimes \mathrm{Id})\psi \bigr) - (Z \otimes \mathrm{Id})\Phi ^+ \otimes \Phi _{\mathrm{junk}} \right\rVert \le \sqrt{2\sqrt{2}\, \delta (\varepsilon )}, \]

and

\[ \left\lVert \mathrm{regSwap}\bigl((V_A \otimes V_B)(A_1 \otimes \mathrm{Id})\psi \bigr) - (X \otimes \mathrm{Id})\Phi ^+ \otimes \Phi _{\mathrm{junk}} \right\rVert \le \sqrt{c_{\mathrm{CHSH}}\varepsilon } + \sqrt{2\sqrt{2}\, \delta (\varepsilon )}. \]

Assume \(\operatorname {Bias}(S) \ge 2\sqrt{2} - \varepsilon \). Then there exist isometries \(V_A\), \(V_B\) and a junk state \(\Phi _{\mathrm{junk}}\) such that

\[ \left\lVert \mathrm{regSwap}\bigl((V_A \otimes V_B)(\mathrm{Id}\otimes B_0)\psi \bigr) - (\mathrm{Id}\otimes H)\Phi ^+ \otimes \Phi _{\mathrm{junk}} \right\rVert \le \sqrt{2\sqrt{2}\, \delta (\varepsilon )}, \]

and

\[ \left\lVert \mathrm{regSwap}\bigl((V_A \otimes V_B)(\mathrm{Id}\otimes B_1)\psi \bigr) - (\mathrm{Id}\otimes ZHZ)\Phi ^+ \otimes \Phi _{\mathrm{junk}} \right\rVert \le \sqrt{c_{\mathrm{CHSH}}\varepsilon } + \sqrt{2\sqrt{2}\, \delta (\varepsilon )}. \]
Remark 23
#

These two theorems are the Lean-facing version of the robust rigidity conclusion in the approximately extremal case: after local isometries, both the state and the designated observables behave like the ideal CHSH realization up to \(O(\sqrt{\varepsilon })\).
