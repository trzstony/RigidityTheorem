1.3 Local Isometries and Exact Extracted Relations

Definition 9 Local extraction isometries
#

Lean declarations

  • Approximate_Rigidity.VA
  • Approximate_Rigidity.VB

From a CHSH strategy \(S\), the Lean development constructs local maps

\[ V_A : H_A \to Qubit \otimes H_A, \qquad V_B : H_B \to Qubit \otimes H_B, \]

using controlled versions of the observables together with a Hadamard rotation on Alice’s side and a \(\pi /8\)-rotation normalization on Bob’s side.

Theorem 11 Exact extracted action of \(A_0\)
#

Uses

  • Definition 9

Lean declarations

  • Approximate_Rigidity.a0_extraction_exact

For every \(\phi \in H_A \otimes H_B\),

\[ (V_A \otimes V_B)(A_0 \otimes \mathrm{Id})\phi = \bigl((Z \otimes \mathrm{Id})\otimes \mathrm{Id}\bigr)(V_A \otimes V_B)\phi . \]

Equivalently, the norm of the difference is exactly zero.

Theorem 12 Exact extracted action of \(B_0\) on the strategy state
#

Uses

Lean declarations

  • Approximate_Rigidity.b0_extraction_exact_on_strategy_state

On the distinguished strategy state \(\psi \),

\[ (V_A \otimes V_B)(\mathrm{Id}\otimes B_0)\psi = \bigl(\mathrm{Id}\otimes (H \otimes \mathrm{Id})\bigr)(V_A \otimes V_B)\psi . \]

The Lean theorem is stated as vanishing norm of the difference on \(\psi \).
