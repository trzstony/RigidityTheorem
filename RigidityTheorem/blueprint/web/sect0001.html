1.1 Setup and basic definitions

Definition 1 Systems and reduced states
#

Lean declarations

  • RigidityToPguess.Systems

Let \(A\) and \(B\) be qubit systems and let \(E\) be an arbitrary quantum system. A tripartite state is a density operator \(\rho _{ABE}\) on \(A\otimes B\otimes E\). We write \(\rho _{AB}:=\operatorname {Tr}_E(\rho _{ABE})\) for the reduced state on \(AB\).

Definition 2 Bell state

The Bell state \(|\Phi ^+\rangle \in A\otimes B\) is

\[ |\Phi ^+\rangle := \frac{|00\rangle + |11\rangle }{\sqrt{2}},\qquad \Phi := |\Phi ^+\rangle \! \langle \Phi ^+|. \]
Definition 3 Canonical CHSH operator on two qubits
#

Uses

  • Definition 1

Lean declarations

  • RigidityToPguess.CHSH

Let \(X,Z\) denote Pauli operators on a qubit. Define observables

\[ A_0:=Z,\quad A_1:=X,\quad B_0:=\frac{Z+X}{\sqrt2},\quad B_1:=\frac{Z-X}{\sqrt2}. \]

The associated CHSH operator on \(A\otimes B\) is

\[ \mathrm{CHSH} := A_0\otimes B_0 + A_0\otimes B_1 + A_1\otimes B_0 - A_1\otimes B_1. \]
Definition 4 Classical–quantum state after Alice’s key measurement
#

Uses

  • Definition 1

Lean declarations

  • RigidityToPguess.rhoXE

Let \(\{ |0\rangle ,|1\rangle \} \) be the computational basis of \(A\), and set \(P_x:=|x\rangle \! \langle x|_A\). From \(\rho _{ABE}\) define subnormalised conditional states

\[ \rho _E^x := \operatorname {Tr}_{AB}\big((P_x\otimes I_{BE})\, \rho _{ABE}\, (P_x\otimes I_{BE})\big),\qquad x\in \{ 0,1\} . \]

The induced classical–quantum state is

\[ \rho _{XE} := \sum _{x\in \{ 0,1\} } |x\rangle \! \langle x|_X\otimes \rho _E^x. \]
Definition 5 Guessing probability
#

Uses

  • Definition 4

Lean declarations

  • RigidityToPguess.pGuess

For the classical–quantum state \(\rho _{XE}\), the guessing probability is

\[ p_{\mathrm{guess}}(X\mid E)_{\rho } :=\sup _{\{ M_0,M_1\} } \sum _{x\in \{ 0,1\} } \operatorname {Tr}(M_x\, \rho _E^x), \]

where the supremum ranges over POVMs on \(E\), i.e. \(M_x\succeq 0\) and \(M_0+M_1=I_E\).
