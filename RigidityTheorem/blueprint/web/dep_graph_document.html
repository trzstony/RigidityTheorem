Definition 2Bell state

The Bell state \(|\Phi ^+\rangle \in A\otimes B\) is

\[ |\Phi ^+\rangle := \frac{|00\rangle + |11\rangle }{\sqrt{2}},\qquad \Phi := |\Phi ^+\rangle \! \langle \Phi ^+|. \]

Definition 3Canonical CHSH operator on two qubits

Let \(X,Z\) denote Pauli operators on a qubit. Define observables

\[ A_0:=Z,\quad A_1:=X,\quad B_0:=\frac{Z+X}{\sqrt2},\quad B_1:=\frac{Z-X}{\sqrt2}. \]

The associated CHSH operator on \(A\otimes B\) is

\[ \mathrm{CHSH} := A_0\otimes B_0 + A_0\otimes B_1 + A_1\otimes B_0 - A_1\otimes B_1. \]

Definition 5Guessing probability

For the classical–quantum state \(\rho _{XE}\), the guessing probability is

\[ p_{\mathrm{guess}}(X\mid E)_{\rho } :=\sup _{\{ M_0,M_1\} } \sum _{x\in \{ 0,1\} } \operatorname {Tr}(M_x\, \rho _E^x), \]

where the supremum ranges over POVMs on \(E\), i.e. \(M_x\succeq 0\) and \(M_0+M_1=I_E\).

Definition 4Classical–quantum state after Alice’s key measurement

Let \(\{ |0\rangle ,|1\rangle \} \) be the computational basis of \(A\), and set \(P_x:=|x\rangle \! \langle x|_A\). From \(\rho _{ABE}\) define subnormalised conditional states

\[ \rho _E^x := \operatorname {Tr}_{AB}\big((P_x\otimes I_{BE})\, \rho _{ABE}\, (P_x\otimes I_{BE})\big),\qquad x\in \{ 0,1\} . \]

The induced classical–quantum state is

\[ \rho _{XE} := \sum _{x\in \{ 0,1\} } |x\rangle \! \langle x|_X\otimes \rho _E^x. \]

Definition 1Systems and reduced states

Let \(A\) and \(B\) be qubit systems and let \(E\) be an arbitrary quantum system. A tripartite state is a density operator \(\rho _{ABE}\) on \(A\otimes B\otimes E\). We write \(\rho _{AB}:=\operatorname {Tr}_E(\rho _{ABE})\) for the reduced state on \(AB\).

Lemma 7Bell basis eigenvalues

Lemma 16Decision-operator form of \(p_{\mathrm{guess}}\)

For a classical–quantum state \(\rho _{XE}\) with \(X\in \{ 0,1\} \),

\[ p_{\mathrm{guess}}(X\mid E)_\rho = \sup _{0\preceq \Pi \preceq I_{XE}} \operatorname {Tr}(\Pi \rho _{XE}), \]

where the supremum is over operators of the form \(\Pi =|0\rangle \! \langle 0|\otimes M_0 + |1\rangle \! \langle 1|\otimes M_1\) with \(\{ M_0,M_1\} \) a POVM on \(E\).

Theorem 19CHSH bound implies guessing bound (no purification)

Let \(\rho _{ABE}\) be a tripartite state with qubit registers \(A,B\) and arbitrary \(E\), and let \(\rho _{AB}=\operatorname {Tr}_E(\rho _{ABE})\). Assume

\[ \varepsilon \le 2\sqrt2-2, \qquad \operatorname {Tr}(\rho _{AB}\, \mathrm{CHSH})\ge 2\sqrt2-\varepsilon , \]

where \(\mathrm{CHSH}\) is the canonical two-qubit operator from Definition 3. Let \(X\) be the computational-basis measurement outcome on \(A\) as in Definition 4. Then

\[ p_{\mathrm{guess}}(X\mid E)_\rho \le \frac12 + \sqrt{\frac{\varepsilon }{2\sqrt2}} + \frac{\varepsilon }{2\sqrt2}. \]

In particular, \(p_{\mathrm{guess}}(X\mid E)_\rho \le \tfrac 12 + O(\sqrt\varepsilon )\).
